# Color Palette

## Source

Colors were ported over from Github's [primer color system](https://primer.style/css/support/color-system)
and adjusted where required.

## Usage

Colors are exposed from `src/theme` as the named export `colors`.

Each palette is then available under the key listed below the name. The numbers below the shades
correspond to the saturation level for the color, with `0` being the lighest, and `9` the darkest.

To access a color with a numbered shade, you must add `00` after the shade number. See
the example below.

```js
colors.bluePalette.blue400 // #2188ff
```
