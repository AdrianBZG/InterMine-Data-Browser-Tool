# Text

## Renders a semantically correct text element

We export a `Text` wrapper that uses [Blueprintjs Text component](https://blueprintjs.com/docs/#core/components/text)
underneath. This is to provide you with a more consistent api that adheres to our design system.

```js
import { Text } from './src/components/Text'
```

### Result

<Preview>
	<Story name="Component">
		<Text fontSize="m2" isMobile={false} fontWeight="regular" lineHeight="default">
			Intermine Data Browser
		</Text>
	</Story>
</Preview>

To use a semantically correct html element, use the `tagName` prop. See below for all the
other available `Text` props.
